Park Tree Mapping Progress with Notebooks

GIS Analysis & Coursework

The embedded ArcGIS Notebook (the ESRI version of the Jupyter Notebook) is a tutorial for the Philadelphia Parks & Recreation (PPR) GIS team to quantify the park trees that have been mapped by the inventory team as of September 2021. The goal is to understand which PPR properties have park trees fully mapped (one point per tree) and which need additional visits.

The end result is a table with a percentage of mapped park trees per PPR property, and this is accomplished by estimating park tree canopy using DBH and using the city's 2018 canopy layer to measure against the estimated park tree canopy. This analysis was completed in an ArcGIS Pro environment using Python.
